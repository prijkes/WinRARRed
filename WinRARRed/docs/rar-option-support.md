# RAR Version Command-Line Option Support

This document shows which command-line options are supported by each RAR version.
Generated by `check_rar_options.py`.

## Quick Reference

### Dictionary Size Options

| Version Range | `-md128` (no suffix) | `-md128k` (k suffix) | `-md4m` (m suffix) |
|---------------|----------------------|----------------------|--------------------|
| RAR 2.x - 3.x | ✓ (2.50+) | ✗ | ✗ |
| RAR 4.x | ✓ | ✓ (2.50+) | ✗ |
| RAR 5.x+ | ✓ | ✓ | ✓ (2.50+) |

### Timestamp Options (-tsm/-tsc/-tsa)

- **First supported version**: RAR 3.20
- Versions before 3.20 silently ignore timestamp options

### Archive Format Options (-ma4/-ma5)

- `-ma4` (force RAR4 format): RAR 5.00+
- `-ma5` (force RAR5 format): RAR 5.00+
- **Important**: RAR 5.50+ defaults to RAR5 format, use `-ma4` to create RAR4 archives
